# Flutter Portfolio App

A simple and responsive Flutter Personal Portfolio App that displays key sections such as Home, About, Education, Projects, Experience, and Contacts.
The app uses smooth scrolling navigation and integrates external links such as GitHub, LinkedIn, Gmail, and Phone Dialer.

This project was completed as an internship task assigned by Apexcify Technologys.

**Features**

Smooth navigation using GlobalKey and Scrollable.ensureVisible
Multiple portfolio sections:
Home
About
Education
Projects
Machine Learning Projects
Experience
Contacts
External link launching using url_launcher
Clean UI using Flutter widgets like Card, ListTile, CircleAvatar
Resume button (Google Drive link)

**Preview**

Project descriptions with GitHub links
Contact options (email, phone, LinkedIn, GitHub)
